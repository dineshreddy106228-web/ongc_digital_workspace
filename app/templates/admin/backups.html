Export Backup

Generate a fresh MySQL dump for the current {{ environment_name }} environment. The file is streamed immediately as .sql.gz and is not retained on the web server.

Runtime requirement: the deployment environment must include mysqldump. On Railway this usually means adding a MySQL client package to the service image.

Restore Procedure

Restore remains terminal-only by design. There is no web restore button yet because full database restores are destructive and not fully rollback-safe.

python -m flask validate-backup --file path/to/backup.sql.gz
python -m flask restore-db --file path/to/backup.sql.gz --confirm-restore
If a restore fails midway, MySQL may already have applied part of the dump. Take a fresh backup before restoring.
